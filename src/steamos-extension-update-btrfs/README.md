# steamos-extension-update-btrfs

Auto-updates steamos-btrfs on schedule.

**SteamOS only** - not compatible with HoloISO.

## Files

### Binaries
- `/usr/sbin/steamos-extension-update-btrfs` - update script

### Systemd Units
- `steamos-extension-update-btrfs.service` - runs updater
- `steamos-extension-update-btrfs.timer` - scheduled trigger
