# steamos-extension-clean-games

Auto-removes untracked game directories and orphaned shader caches from Steam install paths.

## Files

### Binaries
- `/usr/bin/steamos-extension-clean-games` - cleanup script

### Systemd Units (user)
- `steamos-extension-clean-games.service` - runs cleanup
- `steamos-extension-clean-games.timer` - scheduled trigger
